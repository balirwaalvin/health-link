HEALTH LINK MUKONO — App Development Brief

📱 APP OVERVIEW

👥 USER ROLES
Admin (Full Control)
✅ View/edit/delete all patients, visits, clinics
✅ Add/edit/delete clinics
✅ View all system data
✅ Manage staff accounts (optional)
Clinic Staff
✅ View patients list
✅ View patient details & visit history
✅ Request access (sends OTP email)
✅ Add new visits
✅ Register new patients
❌ Cannot edit/delete records
Patient
✅ Receives OTP via email (48291)
✅ Reads code aloud to grant access
❌ No direct login

📱 SCREENS
1. Login
Username + password
Redirect based on role
2. Dashboard
Welcome + quick stats
Navigation cards: Patients, Add Visit, Register
Admin only: "Clinics Management" card
3. Patients List
Search by name or ID
Cards show: name, ID, phone
Tap → Patient Details
Admin only: Edit/delete icons on cards
Floating + button to Register
4. Patient Details
Display: ID, name, phone, email
"🔐 Request Access" button (Blue)
Sends email with OTP 48291
Visit history list (newest first)
Show: diagnosis, clinic, date
"➕ Add Visit" button (Green)
Admin only: Edit/delete buttons
5. Add Visit
Dropdown: Select Patient
Date picker (default today)
Dropdown: Select Clinic
Diagnosis (text)
Prescription (text)
Notes (textarea)
"💾 Save Visit" (Green)
6. Register Patient
Full Name (text)
Phone (text)
Email (email)
Note: display_id auto-generates
"✅ Register" (Green)
7. Clinics Management (Admin only)
List all clinics with edit/delete
"➕ Add Clinic" button
Add/Edit form: name, location, phone
Delete confirmation modal
8. All Visits (Admin only)
Search + filter
List all visits with patient, date, clinic
Edit/delete options
9. About
App info + how it works

📧 OTP EMAIL
Trigger: Click "Request Access"
